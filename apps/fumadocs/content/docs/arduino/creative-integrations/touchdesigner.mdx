TouchDesigner is a node-based visual programming environment for real-time interactive multimedia. Arduino data flows in via serial (USB) or OSC (network), letting you drive visuals, audio, and projections from physical sensors.

## Serial communication with TouchDesigner

Send Arduino data over USB serial and read it in TouchDesigner using a Serial DAT. Parse comma-separated values into channels you can wire to visual parameters.

- [TouchDesigner Interoperability](https://docs.derivative.ca/Interoperability) — Overview of all communication protocols TD supports.
- [Serial DAT](https://docs.derivative.ca/Serial_DAT) — Read serial data from Arduino.
- [OP Snippets](https://docs.derivative.ca/OP_Snippets) — 1,000+ working examples (right-click any operator).
- [TouchDesigner Learning Portal](http://learn.derivative.ca/) — Fundamentals curriculum.

## OSC protocol for network communication

Use the OSC (Open Sound Control) protocol for wireless, higher-bandwidth communication between Arduino (with WiFi) and TouchDesigner.

- [OSC In DAT](https://docs.derivative.ca/OSC_In_DAT) — Receive OSC messages in TouchDesigner.
- [TouchDesigner Forum](https://forum.derivative.ca/) — Community help for Arduino + TD integration.
- [TouchDesigner Official YouTube](https://www.youtube.com/TouchDesignerOfficial) — Tutorials and community showcases.
- [AllTD.org](http://alltd.org/) — Multilingual TouchDesigner learning materials.
