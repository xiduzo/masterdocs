Arduino's serial and network capabilities let it integrate with many creative coding and automation platforms beyond TouchDesigner and Microflow.

## Node-RED

A low-code, flow-based programming tool for wiring together hardware, APIs, and services. Great for IoT dashboards, home automation, and data routing.

- [Node-RED](https://nodered.org/) — Official site with installation guides.
- [Node-RED Arduino Integration](https://nodered.org/docs/hardware/arduino) — Connect Arduino to Node-RED flows.
- [Node-RED on Raspberry Pi](https://nodered.org/docs/hardware/raspberrypi) — Run Node-RED on a Pi alongside Arduino.

<Skill id="arduino-nodered" label="Wire Arduino data into Node-RED flows for IoT automation" />

## Processing

The original creative coding language — and Arduino's sister project. Processing reads serial data from Arduino to create real-time visualizations, generative art, and interactive installations.

- [Processing](https://processing.org/) — Official site with downloads and reference.
- [Getting Started](https://processing.org/tutorials/gettingstarted) — First sketch tutorial.
- [Processing Serial Library](https://processing.org/reference/libraries/serial/index.html) — Read Arduino data over serial in Processing.

<Skill id="arduino-processing" label="Visualize Arduino data in Processing via serial communication" />

## p5.js

The web-based JavaScript version of Processing. Connect Arduino data to browser-based visuals using Web Serial or a Node.js serial bridge.

- [p5.js](https://p5js.org/) — Official site with editor and tutorials.
- [p5.js Examples](https://p5js.org/examples/) — Interactive examples to learn from.
