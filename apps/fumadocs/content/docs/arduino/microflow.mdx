Microflow is an open-source visual flow editor for rapid hardware prototyping. Instead of writing code, you drag and drop nodes to connect sensors, LEDs, servos, and MQTT topics — then deploy directly to your microcontroller.

## Get started with Microflow

Install Microflow, connect your board, and create your first flow. Understand the node-based interface: input nodes (sensors), processing nodes (logic, math), and output nodes (LEDs, servos, MQTT).

- [Microflow Documentation](https://docs.microflow.tech/) — Official docs covering setup, nodes, and deployment.
- [Microflow GitHub](https://github.com/xiduzo/microflow) — Source code, issues, and releases.

## MQTT with Microflow

Microflow has first-class MQTT support. Publish sensor data and subscribe to topics using visual nodes — no PubSubClient code needed.

- [Microflow Documentation](https://docs.microflow.tech/) — MQTT node configuration and broker setup.
