Visual output is one of Arduino's most satisfying capabilities — from a single blinking LED to full graphics on an OLED display.

## Single LEDs and PWM fading

Control individual LEDs with digital pins and fade them smoothly using PWM.

- [Blink](https://docs.arduino.cc/built-in-examples/basics/Blink/) — Turn an LED on and off.
- [Fade](https://docs.arduino.cc/built-in-examples/basics/Fade/) — Smoothly fade an LED with PWM.
- [LED Bar Graph](https://docs.arduino.cc/built-in-examples/display/BarGraph/) — Visualize sensor data as a row of LEDs.

<Skill id="arduino-leds" label="Control LEDs with digital output and PWM fading" />

## LED matrices and strips

Drive addressable LED strips (WS2812 / NeoPixel) and LED matrices for animations and data visualization.

- [Row Column Scanning](https://docs.arduino.cc/built-in-examples/display/RowColumnScanning/) — Control an 8x8 LED matrix.
- [Adafruit NeoPixel Guide](https://learn.adafruit.com/adafruit-neopixel-uberguide) — Comprehensive guide to WS2812B / NeoPixel strips.

<Skill id="arduino-led-strips" label="Drive addressable LED strips and LED matrices" />

## LCD and OLED displays

Display text on a 16x2 LCD or graphics on an OLED screen using I2C or SPI.

- [LCD Displays](https://docs.arduino.cc/learn/electronics/lcd-displays/) — Wire and control character LCD displays.
- [Adafruit SSD1306 OLED Guide](https://learn.adafruit.com/monochrome-oled-breakouts) — Popular 128x64 OLED display setup.
