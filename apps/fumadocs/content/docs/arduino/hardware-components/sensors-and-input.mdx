## Buttons and potentiometers

Read digital inputs from pushbuttons and analog inputs from potentiometers to control your sketches.

* [Button](https://docs.arduino.cc/built-in-examples/digital/Button/) — Wire and read a pushbutton.
* [Potentiometer Basics](https://docs.arduino.cc/learn/electronics/potentiometer-basics/) — Read a variable resistance as analog input.
* [Analog Read Serial](https://docs.arduino.cc/built-in-examples/basics/AnalogReadSerial/) — Print potentiometer values to the Serial Monitor.

## Distance and motion sensors

Measure distance with ultrasonic sensors and detect movement with PIR motion sensors.

* [Ping — Ultrasonic Range Finder](https://docs.arduino.cc/built-in-examples/sensors/Ping/) — Measure distance using an ultrasonic sensor.
* [Knock Sensor](https://docs.arduino.cc/built-in-examples/sensors/Knock/) — Detect vibration and taps with a piezo.
* [Adafruit Sensor Tutorials](https://learn.adafruit.com/) — 3,300+ guides including DHT temperature, PIR motion, and light sensors.
