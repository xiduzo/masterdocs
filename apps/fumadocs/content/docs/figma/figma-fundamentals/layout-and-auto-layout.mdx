Auto Layout is Figma's equivalent of CSS Flexbox — it lets frames automatically arrange, space, and resize their children. It's the single most important layout feature in Figma.

## Constraints and positioning

Control how elements respond when their parent frame is resized — pin to edges, center, or scale proportionally.

- [Constraints](https://help.figma.com/hc/en-us/articles/360039957734) — Pin elements to edges or center them within a frame.

## Resizing behavior

Choose how elements size themselves: **hug** (shrink to content), **fill** (stretch to available space), or **fixed**. Add min/max dimensions for responsive bounds.

- [Grid flow in Auto Layout](https://help.figma.com/hc/en-us/articles/31289469907863) — Build galleries and dashboards with the grid flow.
- [Grid playground](https://www.figma.com/community/file/1484548529005244626) — Practice with grid Auto Layout.
