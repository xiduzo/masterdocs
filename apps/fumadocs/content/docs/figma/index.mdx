This learning path takes you from zero Figma experience to connecting your designs directly to AI coding agents. Each topic gives you a brief overview then points you to the best official resources — Figma's own courses, docs, and videos.

## What you will learn

### Figma Fundamentals

Get comfortable with the workspace, core tools (frames, shapes, text, images), and Auto Layout — Figma's most important layout feature.

### Components and Design Systems

Build reusable components with variants and properties, implement design tokens using variables and modes, then bring it all together as a published team library.

### Development and AI

Bridge design and code with Dev Mode, Code Connect, and the Figma REST API. Then connect AI coding agents to your designs using the Figma MCP server and explore prompt-to-app with Figma Make.

### Alternatives

Explore the design tool landscape beyond Figma — open-source options like Penpot, code-native tools like Paper, and other platforms worth knowing about.
