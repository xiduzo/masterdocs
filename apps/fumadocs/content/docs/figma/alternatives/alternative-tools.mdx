Figma dominates the design tool space, but several alternatives offer unique advantages — open-source licensing, code-native canvases, or tighter AI integration. Knowing what's out there helps you pick the right tool for a project.

## Penpot — open-source and self-hostable

Penpot is a free, open-source design and prototyping platform. It uses SVG natively (not proprietary formats), runs in the browser, and can be self-hosted for full data ownership. Great for teams that need open-source licensing or on-premise deployments.

- [Penpot](https://penpot.app/) — Official site with web app, downloads, and features overview.
- [Penpot User Guide](https://help.penpot.app/) — Official documentation covering workspace, components, design tokens, and prototyping.
- [Penpot GitHub](https://github.com/penpot/penpot) — Source code and self-hosting instructions.
- [Penpot YouTube](https://www.youtube.com/@paborat) — Tutorials and feature walkthroughs.

## Paper — code-native canvas with MCP

Paper is a design tool built on web standards (HTML/CSS) where your canvas output is real code. It features a desktop app that connects to AI coding agents (VS Code, Cursor, Claude) via its own MCP server, creating a continuous loop between design and codebase.

- [Paper](https://paper.design/) — Official site with features and downloads.
- [Paper Docs](https://paper.design/docs) — Getting started and feature guides.
- [Paper MCP Server](https://paper.design/docs/mcp) — Connect AI agents to your design files via MCP.
- [Paper vs Figma](https://paper.design/compare/figma) — Official comparison between Paper and Figma.
- [Paper YouTube](https://www.youtube.com/@paper-design-tutorials) — Tutorials and demos.

## Other tools to know about

The design tool landscape keeps moving. These are worth knowing about depending on your workflow.

- [Framer](https://www.framer.com/) — Visual site builder that publishes directly to the web. Blurs the line between design tool and website builder.
- [Webflow](https://webflow.com/) — Visual web development platform. Produces production HTML/CSS without code, includes CMS and hosting.
- [Sketch](https://www.sketch.com/) — macOS-native design tool that predates Figma. Still used by some teams, especially in the Apple ecosystem.
- [Lunacy](https://icons8.com/lunacy) — Free design tool by Icons8. Supports Sketch files natively and includes built-in assets.
