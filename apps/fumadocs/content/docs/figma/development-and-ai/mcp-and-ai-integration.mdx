The Figma MCP server connects AI coding tools (VS Code, Cursor, Claude, Codex) directly to your Figma designs using the Model Context Protocol. Beyond reading designs, the remote server can also write to the canvas.

## Understand how MCP bridges AI and Figma

Learn what the MCP server provides: layout, colors, variables, Code Connect mappings, and screenshots — all fetched from a Figma URL in your AI prompt.
