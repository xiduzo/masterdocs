Dev Mode is Figma's dedicated developer view showing measurements, code snippets (CSS, iOS, Android), variable names, and export settings — without the noise of design tools.

## Navigate Dev Mode and inspect designs

Switch to Dev Mode, filter for "Ready for development" sections, and inspect CSS properties, spacing, and variable names.

- [Dev Mode overview](https://help.figma.com/hc/en-us/articles/15023124644247) — What Dev Mode is and how to use it.
- [Inspect designs in Dev Mode](https://help.figma.com/hc/en-us/articles/360055203533) — View CSS, measurements, spacing, and variable names.
- [Variables in Dev Mode](https://help.figma.com/hc/en-us/articles/27882809912471) — Access variable details and the variables table.
- [Compare changes in Dev Mode](https://help.figma.com/hc/en-us/articles/17916297659543) — Diff design versions to see what changed.
- [Dev Mode fundamentals](https://help.figma.com/hc/en-us/articles/35498519152663) — From the Figma MCP collection.

## Mark ready and export assets

Use the ready-for-development workflow to signal handoff. Export icons, illustrations, and assets in PNG, SVG, JPG, or PDF at multiple scales.

- [Mark designs as ready for development](https://help.figma.com/hc/en-us/articles/15023220813335) — Ready-status workflow between designers and developers.
- [Export assets from designs](https://help.figma.com/hc/en-us/articles/360040028114) — Export in PNG, SVG, JPG, PDF at multiple scales.

## Code Connect

Map Figma components to your actual codebase components. Instead of generic CSS, developers see the real JSX/Swift/Compose code they should use.

- [Code Connect](https://help.figma.com/hc/en-us/articles/23920389749655) — Map Figma components to your codebase components.
- [Code Connect GitHub repo](https://github.com/figma/code-connect) — Set up mappings for React, Swift, Compose, and more.
- [Dev Mode MCP server with Code Connect](https://help.figma.com/hc/en-us/articles/35498566189335) — Combining Dev Mode, Code Connect, and MCP.

<Skill id="figma-code-connect" label="Map Figma components to codebase components with Code Connect" />

## Figma REST API

Automate token extraction, asset export, and design data queries using the Figma APIs.

- [Figma REST API](https://www.figma.com/developers/api) — Programmatic access to files, components, styles, variables, images, and webhooks.
- [Plugin API: Working with variables](https://www.figma.com/plugin-docs/working-with-variables) — Build plugins that create, read, and bind variables.
