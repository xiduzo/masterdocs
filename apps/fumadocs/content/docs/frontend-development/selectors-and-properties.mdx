CSS selectors let you target specific HTML elements for styling. Element selectors, class selectors, and ID selectors are the most common. Combinators like descendant and child selectors give you finer control over which elements receive styles.

Properties define the visual appearance of selected elements. Common properties include `color`, `font-size`, `margin`, and `padding`. Understanding specificity helps you predict which styles win when multiple rules target the same element.
