This roadmap takes you from the fundamentals of web development through to building modern, interactive user interfaces. It's designed to be followed in order, but you can jump to any topic that interests you.

## What you'll learn

The roadmap is split into tracks that build on each other:

### JavaScript Fundamentals

Get comfortable with the language that powers the web. You'll cover variables, data types, functions, and scope — the building blocks for everything that follows.

### CSS Basics

Learn how to style and lay out web pages. From selectors and properties to the box model and layout techniques, this track gives you the visual toolkit you need.

## How to follow this roadmap

1. Start with the first topic in JavaScript Fundamentals
2. Work through each topic in order, checking off skills as you go
3. Move on to CSS Basics once you're comfortable with the JS fundamentals
4. Use the prev/next links at the bottom of each topic to stay on track

You can also view your progress on the [roadmap overview page](/roadmaps/frontend-development).
