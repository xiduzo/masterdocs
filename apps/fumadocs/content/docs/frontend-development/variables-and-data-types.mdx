JavaScript provides three ways to declare variables: `let`, `const`, and `var`. Each has different scoping rules and use cases. Modern JavaScript favors `let` for reassignable variables and `const` for constants.

Primitive data types include `string`, `number`, `boolean`, `null`, `undefined`, `symbol`, and `bigint`. Understanding these types is essential for writing correct JavaScript.
