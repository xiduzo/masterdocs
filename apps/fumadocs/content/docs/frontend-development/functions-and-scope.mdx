Functions are the building blocks of JavaScript programs. You can define them as declarations, expressions, or arrow functions. Each form has subtle differences in hoisting and `this` binding.

Scope determines where variables are accessible. JavaScript uses block scope for `let` and `const`, and function scope for `var`. Closures allow inner functions to access variables from their outer scope even after the outer function has returned.
